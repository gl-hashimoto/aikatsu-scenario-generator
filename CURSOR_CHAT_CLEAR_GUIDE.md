# Cursor 会話履歴クリア方法ガイド

## 🎯 最も簡単な方法

### 方法1: チャットパネル内のメニューからクリア

1. **チャットパネルの右上角を確認**
   - 「…」（三点メニュー）アイコンまたは「×」アイコンがあるはずです

2. **メニューをクリック**
   - 「Clear Chat」または「チャットをクリア」を選択
   - または「Reset Chat」を選択

3. **確認ダイアログが表示されたら「OK」をクリック**

### 方法2: 新しいチャットを開始（推奨）

1. **左サイドバーの「+ New Chat」ボタンをクリック**
   - または `Cmd+L`（Mac）/ `Ctrl+L`（Windows）を押す

2. **新しいチャットが開きます**
   - これで実質的にクリアされた状態になります
   - 問題のある画像は含まれていません

## 📋 その他の方法

### 方法3: コマンドパレットから

1. **コマンドパレットを開く**
   - `Cmd+Shift+P`（Mac）/ `Ctrl+Shift+P`（Windows）

2. **「Clear Chat」と入力**
   - 「Clear Chat History」または「チャットをクリア」を選択

3. **実行**

### 方法4: 設定からクリア

1. **設定を開く**
   - `Cmd+,`（Mac）/ `Ctrl+,`（Windows）
   - またはメニューから「Settings」→「Preferences」

2. **「Cursor」または「Chat」セクションを探す**

3. **「Clear Chat History」または「チャット履歴をクリア」を選択**

## 🔍 画面の見つけ方

### チャットパネルの場所

- **右側パネル**: 多くの場合、エディタの右側にチャットパネルが表示されます
- **下部パネル**: 場合によっては、エディタの下部に表示されることもあります
- **左サイドバー**: チャット履歴の一覧が表示されることもあります

### メニューの場所

- **右上角**: チャットパネルの右上に「…」や「×」アイコン
- **チャットタイトル**: チャットのタイトルを右クリックするとメニューが表示されることもあります

## ⚠️ 注意事項

1. **会話履歴をクリアすると、以前の会話内容は復元できません**
   - 重要な内容がある場合は、先にコピーしてください

2. **新しいチャットを開始する方が安全**
   - 既存のチャットを削除する必要はありません
   - 新しいチャットで問題が解決します

3. **画像エラーの場合**
   - 問題のある画像が会話履歴に含まれているため
   - 新しいチャットを開始することを強く推奨します

## 🚀 推奨手順（画像エラーが発生している場合）

1. **左サイドバーの「+ New Chat」をクリック**
2. **新しいチャットが開いたことを確認**
3. **必要に応じて、テキストのみで質問を再送信**
4. **画像を送信する場合は、事前に2000ピクセル以下にリサイズ**

## 💡 ショートカット一覧

- **新しいチャット**: `Cmd+L`（Mac）/ `Ctrl+L`（Windows）
- **コマンドパレット**: `Cmd+Shift+P`（Mac）/ `Ctrl+Shift+P`（Windows）
- **設定**: `Cmd+,`（Mac）/ `Ctrl+,`（Windows）

## 🆘 それでも見つからない場合

1. **Cursorのバージョンを確認**
   - メニューから「Help」→「About」でバージョンを確認
   - 最新版にアップデートしてみてください

2. **Cursorを再起動**
   - 完全に終了してから再度起動
   - 新しいチャットを開始

3. **画面のレイアウトを確認**
   - チャットパネルが非表示になっている可能性があります
   - `Cmd+L`（Mac）/ `Ctrl+L`（Windows）でチャットを開く
